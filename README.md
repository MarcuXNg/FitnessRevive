### Install

`npm install` to install all packages

### Run

`npm start` to run

### Clone

`git clone --recursive -b [branchname] [url]` to clone

### update to repo

`git add .`, `git commit -m 'message'` and `git push` in each subfolder to push to repo

### git submodule add

`git submodule add <gitUrl> <FolderName>`
